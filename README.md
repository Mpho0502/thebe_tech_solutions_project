## 🗄️ Data Model Foundation
The company’s operations are supported by a robust **star schema**, designed to capture and analyze every aspect of its business:

- **Fact Table – Sales_Fact:** Tracks transactions, revenue, discounts, expenses, and profits.  
- **Dimensions:**  
  - **Employee** → Performance and productivity of staff  
  - **Clients** → Geographic and demographic insights into the customer base  
  - **Accounts** → Service contracts and solution packages  
  - **Product** → Categories and subcategories of tech/security offerings  
  - **Date** → Time‑based analysis for trends and forecasting  

This schema ensures Thebe Tech Solutions can monitor **profitability, client engagement, and operational efficiency** with precision.
